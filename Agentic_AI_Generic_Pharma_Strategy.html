Slide 6

Two flagship agents

Build these first. They sell the rest of the product.
AGENT 1
Patent Certification
Highest stakes. Highest defensibility.
WHAT IT DOES
  • For each Orange Book patent on the RLD, recommends Para I/II/III/IV or §viii carve-out
  • Drafts Module 1.3.5.1 + 1.3.5.2 documents
  • Generates Para IV Notice Letter when applicable
  • Estimates 180-day first-to-file exclusivity value
REPLACES TODAY
IP attorney + Regulatory Director + 2–4 weeks on a 12-patent RLD. $40K–$80K outsourced; agent does it overnight.
MOAT: Fine-tuned on FDA Para IV history + use code register
AGENT 2
Labeling Comparison
Highest immediate time savings. Lowest legal risk.
WHAT IT DOES
  • Pulls latest RLD label (SPL XML) from Drugs@FDA
  • Section-by-section diff: token + semantic
  • Categorizes differences (permitted / carve-out / error)
  • Drafts justification narratives for each difference
  • Verifies §viii carve-out completeness across label
REPLACES TODAY
Senior regulatory writer, 3–5 days per label. Error-prone — humans miss orphan references.
MOAT: SPL-native + structured FDA IID + carve-out QA
Synergy: §viii decisions from Agent 1 feed directly into carve-out completeness checks in Agent 2.

Risks & mitigations

What can kill this — and how to handle it
Hallucinations on regulatory output
Incorrect Para IV recommendation could trigger litigation or lose exclusivity.
Mitigation: Position as 'draft for human review.' Always require RA + IP counsel sign-off.
Slow procurement at small pharma
Even small companies have 8–14 week procurement cycles for new vendors.
Mitigation: Free Orange Book monitor as land-step. Build relationships first.
FDA changing the rules
eCTD v4.0 transition, Use Code reform, AI guidance evolving.
Mitigation: Configurable knowledge base. Quarterly review with regulatory advisor.
Big incumbents copying the playbook
Veeva, IQVIA, or FDA-adjacent CROs could build similar agents.
Mitigation: Move fast. Win the trust contract. Stay narrowly vertical.
Customer data sensitivity
Customer regulatory dossiers are highly confidential pre-filing.
Mitigation: Single-tenant from day one. SOC 2 Type II within year one.
Validation creep
Customers may demand Part 11 / GxP validation before purchase.
Mitigation: Stay 'drafting tool' positioning v1. Validated mode for v2.
